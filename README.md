# 📊 Covid-19 Pandemic Analytics using Python

## 🔹 Overview
This project focuses on analyzing Covid-19 pandemic data using Python. It involves data cleaning, exploratory data analysis (EDA), visualization, and basic machine learning techniques to identify trends and patterns in the data.

---

## 🔹 Objectives
- Analyze Covid-19 data to understand trends
- Perform data cleaning and transformation
- Visualize confirmed, recovered, and death cases
- Apply basic machine learning and predictive techniques

---

## 🔹 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Anaconda / Jupyter Notebook  

---

## 🔹 Key Features

### ✅ Data Processing
- Loaded multiple datasets
- Cleaned and transformed data
- Removed unnecessary columns
- Filtered state-wise data (e.g., Rajasthan)

### ✅ Exploratory Data Analysis (EDA)
- Analyzed trends of confirmed, recovered, and death cases
- Performed correlation analysis
- Created derived metrics such as:
  - Mortality Rate
  - Recovery Rate

### ✅ Data Visualization
- Line charts for trend analysis  
- Pie charts for distribution  
- Heatmaps for correlation analysis  

### ✅ Machine Learning
- Implemented **K-Nearest Neighbors (KNN)** classifier  
- Performed:
  - Data preprocessing
  - Label encoding
  - Feature scaling
  - Model evaluation  

### ✅ Predictive Analysis
- Applied **curve fitting (sigmoid function)** for trend prediction  
- Implemented **SIR Model (Susceptible-Infected-Recovered)** to simulate pandemic spread
